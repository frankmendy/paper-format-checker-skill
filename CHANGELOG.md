# 更新日志 (Changelog)

所有版本的显著变更都将记录在此文件中。

## [2.0] - 2025-04-17

### 🎯 重大更新

#### 1. 修复二、三级标题段前段后"自动"配置问题
- **问题**: 用户要求二、三级标题段前段后为"自动"，但多次修复后 Word 中仍显示为固定值（如段后10磅）
- **解决方案**: 
  - 发现 Word 中"自动"的真正含义：`before="100" beforeAutospacing="1"` (5磅自动)
  - 使用底层 XML 精确设置段前段后自动配置
  - 添加保存前最终清理和保存后验证机制

#### 2. 添加依赖检查机制
- 在程序启动时自动检查必要的依赖包
- 如果缺少 `python-docx` 或 `pywin32`，会显示友好的安装提示
- 避免因缺少依赖导致的修复失败

#### 3. 更新格式规则文档
- 添加"排坑经验（二、三级标题段前段后"自动"的真正含义）"
- 记录正确的 XML 配置方法和解决方案

### 🔧 技术改进

- **新增函数**:
  - `_set_paragraph_spacing_auto()`: 按照 Word 实际配置设置段前段后自动
  - `_verify_heading_spacing_auto()`: 验证标题段前段后是否为正确的自动配置
  - `_final_cleanup_headings()`: 保存前最终清理确保配置正确
  - `_verify_after_save()`: 保存后重新读取验证配置
  - `check_dependencies()`: 启动时依赖检查

- **改进验证逻辑**:
  - 复检阶段检查 XML 中的实际配置
  - 保存后验证确保配置正确落盘

### 📚 文档更新

- **SKILL.md**: 添加 Prerequisites / Dependencies 章节
- **FORMAT_RULES_LEDGER.md**: 添加新的排坑经验
- **CHANGELOG.md**: 创建版本更新日志

### 🐛 修复问题

- 修复二、三级标题段前段后配置被样式默认值覆盖的问题
- 修复 Word 保存后段前段后配置丢失的问题
- 修复依赖缺失时程序报错不友好的问题

## [1.0] - 初始版本

### ✨ 功能特性

- 自动修复论文格式，匹配"网络空间安全学院论文格式模板"
- 三阶段修复流程：
  1. 标题与摘要修复
  2. 目录修复
  3. 正文与题注修复
- 支持 Word 自动目录更新
- 循环复检机制确保修复质量
- 详细的修复报告生成

### 📋 支持的格式规则

- 页边距设置
- 中英文摘要格式
- 关键词格式
- 目录格式（三级目录）
- 正文格式（三级标题）
- 图表题注格式
- 参考文献格式
- 致谢、附录等特殊章节格式
